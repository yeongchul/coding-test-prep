## 🐳 코딩 테스트 문제 풀이 저장소
이 레포지토리는 백준, 프로그래머스 등에서 푼 문제들을 정리한 저장소입니다.
<br>
풀이한 문제는 Java로 작성되었으며 문제별로 정리되어 있습니다.

<br>

### 👩🏻‍🔧 사용 언어
Java

<br>

### ✏️ 커밋 메시지 규칙
커밋 메시지는 다음 형식을 따릅니다:
<br>
`solve: [플랫폼] [문제번호] - [문제명]`

예시:
- `solve: 백준 9012 - 괄호`

<br>
